More on Styling Using CSS

CSS Variables

Same concept with variables in programming. The value inside a variable can be reused in for any elements

Should be defined inside the :root selector. Example:

Defining Variables

            :root {
                --bg-white: #fff;
                --bg-dark: #000;
                --color-black: #2f2f2f;
                --color-danger: rgb(255, 0, 0);
                --color-warning: rgb(0, 255, 255);
            }

Using Variables

            .u-bg-white {
                background-color: var(--bg-white);
            }
            
            .u-bg-dark {
                background-color: var(--bg-dark);
            }

CSS Selectors

The > 'children' Selector

Select any immediate children inside an element

The + 'adjacent sibling' Selector

Select ONE element directly placed after one another specific element

The ~ 'general sibling' Selector

Select ANY elements placed after one another specific element

Making elements responsive to any devices
